# PL/SQL Exam Recap

Streamlit app with short PL/SQL exam recap pages: definitions, syntax, slide-based examples, and self-check lists.

The app is self-contained for deployment. The original PowerPoint source files are not required.

https://plsql-overview-recap.streamlit.app/

## Run

```powershell
pip install -r requirements.txt
streamlit run app.py
```
